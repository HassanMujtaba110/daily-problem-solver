# Daily Problem Solver Website

A responsive static website for the Daily Problem Solver project featuring daily math puzzles and coding challenges.

## Features

- Clean, modern design using Poppins font
- Professional color palette with blue (#2D9CDB), green (#27AE60), light gray background (#F9FAFB), and dark text (#333333)
- Home page with "Math Puzzle of the Day" and "Coding Challenge of the Day"
- Archive page listing previous problems
- Problem detail page with social sharing functionality
- Fully responsive layout for mobile and desktop

## Files Included

- `index.html` - Home page with daily problems
- `archive.html` - Archive page with past problems
- `problem-detail.html` - Detailed view of specific problems
- `styles.css` - All styling for the website
- `script.js` - JavaScript functionality for interactivity
- `README.md` - This file

## How to Use

1. Simply open `index.html` in any modern web browser
2. Navigate between pages using the top navigation menu
3. On the home page:
   - Click "Reveal Answer" to show the solution to the math puzzle
   - Click "Reveal Hint" to show the hint for the coding challenge
4. Visit the Archive page to browse past problems
5. Click on any archived problem to see its details
6. Use social sharing buttons on problem detail pages to share with friends

## Customization

To customize the problems or content:
1. Edit the HTML files directly to change problem statements
2. Modify `styles.css` to adjust colors, fonts, or layout
3. Update `script.js` to change functionality

## Browser Compatibility

The website works on all modern browsers (Chrome, Firefox, Safari, Edge).

## Responsive Design

The website is fully responsive and adapts to different screen sizes:
- Desktop: Full layout with sidebar navigation
- Tablet: Adjusted spacing and sizing
- Mobile: Collapsible navigation menu and optimized layout
