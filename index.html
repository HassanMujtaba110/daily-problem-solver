Math Puzzle of the Day

Find the Missing Number

January 10, 2026

What is the next number in the sequence: 2, 6, 12, 20, 30, ?

Answer:

The pattern is n(n+1) where n starts from 1: 1×2=2, 2×3=6, 3×4=12, 4×5=20, 5×6=30, so the next is 6×7=42.

Coding Challenge of the Day

Two Sum Problem

January 10, 2026

Given an array of integers nums and an integer target, return indices of the two numbers such that they add up to target.

Example:

Input: nums = [2,7,11,15], target = 9

Output: [0,1]

Explanation: Because nums[0] + nums[1] == 9, we return [0, 1].

Hint:

Use a hash map to store the difference between the target and each number as you iterate through the array. This allows you to find the complement in O(1) time.
