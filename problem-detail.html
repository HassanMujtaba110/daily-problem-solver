Sequence Pattern Recognition

January 9, 2026

Problem Statement

What is the next number in the sequence: 1, 4, 9, 16, 25, ?

Solution

The sequence represents perfect squares: 1², 2², 3², 4², 5², so the next number is 6² = 36.
